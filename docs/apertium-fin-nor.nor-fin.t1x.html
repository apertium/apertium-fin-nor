    Rules

    The actual rules concerning stuff.

    adp art adj noun

    Matching pattern:

    1. adp
    2. art
    3. adj
    4. noun

    Action:

    1. adp-mangler( $1 )
    2. comp-mangler( $3 )
    3. number-mangler( $4 )
    4. Output:
      1. [
        1. tl[3]['lem'] tl[3]['a_adj'] $comparison $number $adpcase
        2. b2
        3. tl[4]['lem'] tl[4]['a_noun'] $number $adpcase
        4. b1
        5. $maybeadp
        ]adpnoun <NP> $adpcase
